# React Router Advanced

A React project demonstrating advanced routing techniques using React Router. This project includes nested routes, dynamic routes, and protected routes that require authentication.

## Features

- Home page
- Nested routes for Profile component (ProfileDetails and ProfileSettings)
- Dynamic routing for blog posts
- Protected routes that require login
- Simple authentication simulation
- Built with React and Vite
