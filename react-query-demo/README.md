
# React Query Demo

A React project demonstrating advanced data fetching, caching, and updating using React Query (v5). This project fetches posts from the JSONPlaceholder API and showcases how React Query handles loading states, error handling, caching, and manual data refetching.

## Features

- Fetch posts from JSONPlaceholder API
- Display loading and error states
- Cache data automatically using React Query
- Refetch data manually with a button
- Built with React and Vite
